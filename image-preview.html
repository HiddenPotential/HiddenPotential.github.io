Image Preview Tool

This page helps you preview your images before adding them to your homepage.

When you add images to the images folder, update the file paths below to see how they'll look.

Profile Image

Update the src attribute to point to your profile image:

File path: images/profile.jpg

This will be displayed as a circle on your homepage.

Roblox Image

Update the src attribute to point to your Roblox image:

File path: images/roblox.jpg

Golf Image

Update the src attribute to point to your golf image:

File path: images/golf.jpg
